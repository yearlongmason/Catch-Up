get in the same directory as this text file and use the command 

python manage.py runserver

to run the tailwind compiler (needed when updating the css) run the command 

./tailwindcss -i ./sim/static/css/input.css -o ./sim/static/css/output.css --watch

 but replace the path of ./tailwindcss with the correct path

download tailwind here https://github.com/tailwindlabs/tailwindcss/releases/tag/v3.4.1
